# Adorbis Workspace Packs

Configuration files that connect open-source AI workspace tools to Adorbis AI.

## What is a Starter Pack?

Each folder contains a ready-to-use configuration file for the named workspace tool.
Copy the file into the tool's config directory, replace `YOUR_ADORBIS_API_KEY` with your key, and you are connected to 40+ AI models through Adorbis.

## Quick Setup (Any Tool)

Most tools that support a custom OpenAI-compatible endpoint work with these three values:

- Base URL: https://api.adorbistech.com/v1
- API Key: YOUR_ADORBIS_API_KEY
- Model: adorbis/auto
