## Quick Start

**Step 1 — Get your free API key**

Sign up at [ai.adorbistech.com](https://ai.adorbistech.com) — 1,000 free Adorbis Credits included. No credit card required.

**Step 2 — Clone this repo**

```bash
git clone https://github.com/adorbistech/workspace-packs.git
```

**Step 3 — Copy the config for your workspace**

```bash
# Example: AnythingLLM
cp workspace-packs/anythingllm/config.json ~/path/to/anythingllm/config.json
```

**Step 4 — Replace your key**

Open the config file and replace `YOUR_ADORBIS_API_KEY` with your key from [ai.adorbistech.com](https://ai.adorbistech.com).

**Step 5 — Start your workspace**

That's it. Your workspace is now connected to 40+ AI models through Adorbis.

## Works With Any OpenAI-Compatible Tool

If your tool supports a custom OpenAI-compatible endpoint, use these three values:

```
Base URL:  https://api.adorbistech.com/v1
API Key:   YOUR_ADORBIS_API_KEY
Model:     adorbis/auto
```
